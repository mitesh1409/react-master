# Module #2: JavaScript Fundamentals for React

## Destructuring in JavaScript (Object & Array)

Topics

- Object Destructuring
- Array Destructuring

---

Here are simple examples of each:

## Object Destructuring

Extract properties from an object into variables:

```javascript
const person = {
  name: "Bob",
  age: 30,
  city: "London"
};

// Extract properties into variables
const { name, age } = person;
console.log(name);  // "Bob"
console.log(age);   // 30

// Rename variables while destructuring
const { name: fullName, city: location } = person;
console.log(fullName);  // "Bob"
console.log(location);  // "London"

// Provide default values
const { country = "UK" } = person;
console.log(country);  // "UK" (default, since country doesn't exist)
```

## Array Destructuring

Extract values from an array into variables:

```javascript
const colors = ["red", "green", "blue", "yellow"];

// Extract values into variables
const [first, second] = colors;
console.log(first);   // "red"
console.log(second);  // "green"

// Skip elements
const [, , third] = colors;
console.log(third);   // "blue"

// Use rest operator to capture remaining items
const [primary, ...others] = colors;
console.log(primary);  // "red"
console.log(others);   // ["green", "blue", "yellow"]

// Swap variables easily
let a = 1;
let b = 2;
[a, b] = [b, a];
console.log(a);  // 2
console.log(b);  // 1
```

# Module #2: JavaScript Fundamentals for React

## Classes, Objects, and Getters/Setters in JavaScript

Topics

- Classes
- Objects
- Constructor
- Getters & Setters
- Convention -> attributes starting with "_" should be treated as private, we should not access it directly, that is why we write Getters & Setters for it.

---

Example

```JavaScript
class LibraryItem {
    constructor(title, itemId) {
        this._title = title;
        this._itemId = itemId;
        this._isBorrowed = false;
    }

    get title() {
        console.log('get title called');
        return this._title;
    }

    set title(value) {
        console.log('set title called');
        this._title = value;
    }
}

const libraryItem = new LibraryItem('ABCD', 1001);
console.log(libraryItem);
console.log(libraryItem.title);
console.log(libraryItem._itemId);
console.log(libraryItem._isBorrowed);

libraryItem.title = 'EFGH';
console.log(libraryItem);
```

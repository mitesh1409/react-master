---

Example: `splice()`  

```javascript
const numbers = [10, 20, 30, 40, 50];

// Remove/Delete
// Remove all elements starting from index 1.
numbers.splice(1);

// Remove 1st, 2nd and 3rd elements from the list.
numbers.splice(0, 3);

// Remove 3 elements starting from index 2.
numbers.splice(2, 3);

// Replace/Update
// Replace first two elements with 100, 200 respectively.
numbers.splice(0, 2, 100, 200);

// Insert
// Add 3 elements 99, 88, 77 starting from index 3.
numbers.splice(3, 0, 99, 88, 77);
```

Example: `slice()`  

```javascript
const numbers = [10, 20, 30, 40, 50];

// Get first 3 elements.
const sliceOfFirst3 = numbers.slice(0, 3);

// Get a full slice OR shallow copy.
const fullSlice = numbers.slice();
```
